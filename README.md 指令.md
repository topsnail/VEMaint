## 🐛 常见问题排查
1. 本地启动失败：排查依赖、端口占用、Node版本
2. CF Pages部署失败：构建命令错误、目录错误、资源绑定失败
3. 页面访问异常：404/522/530错误排查、缓存刷新、Headers配置
4. 服务端函数（Functions）不生效：文件路径、命名、权限排查
5. 环境变量不生效：配置位置、重启部署、变量名大小写问题
